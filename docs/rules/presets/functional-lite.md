# 🟢 Functional Lite

Use this preset for low-friction functional constraints.

## Config key

```ts
immutable.configs["functional-lite"]
```

## Flat Config example

```ts
import immutable from "eslint-plugin-immutable-2";

export default [immutable.configs["functional-lite"]];
```

This preset is a practical on-ramp before `functional` or `all`.
